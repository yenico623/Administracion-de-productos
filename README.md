# 📝 Administracion de productos 
Aplicativo web desarrollado con JavaScript para la gestion de tarjetas.

## ✅ Funcionalidades
- Visualizar tarjetas de productos con imagen, nombre y descripción
- Agregar nuevos productos desde formulario
- Guardar los productos en localStorage

## 🌐 Tecnologías Usadas 
- HTML5
- CSS3
- JavaScript
